Para quem tem um vﾃｭdeo de 9 horas de duraﾃｧﾃ｣o e gostaria de fazer um vﾃｭdeo acelerado resumindo o video original, segue a soluﾃｧﾃ｣o usando ffmpeg:

Agora, entenda a linha:

-vf ﾃｩ a opﾃｧﾃ｣o para habilitar o “Video Filter”. Para o parametro setpts vocﾃｪ deve passar a taxa de aceleraﾃｧﾃ｣o. Para fazer o “timelapse”, o ffmpeg irﾃ｡ descartar uma sﾃｩrie de frames, e com isso ﾃｩ possﾃｭvel gerar um vﾃｭdeo “acelerado”.

No exemplo que usei acima, um video de 20 minutos ﾃｩ acelerado para um de apenas 1 minuto. Portanto o calculo: 1/20=0.05

Para acelerar um vﾃｭdeo de 9 horas para 3 minutos vocﾃｪ teria que calcular da seguinte forma:
3/(9*60)=0.00555, entﾃ｣o o parametro ficaria: “-vf setpts=0.00555*PTS”

Editado em 04/dez/2013:

Abaixo segue um exemplo simples que faz isso, usando FFmpeg:

ffmpeg -i teste.avi -pix_fmt rgb24 -f gif teste.gif

Detalhes:

  • -i = especifica o arquivo de entrada (teste.avi)
  • -pix_fmt = especifica o formato dos pixels (rgb24, formato suportado pelo gifﾂ )
  • -f = forﾃｧa a saida para .gif
  • teste.gif ﾃｩ o arquivo de saﾃｭda do comando

Se vocﾃｪ precisa de uma soluﾃｧﾃ｣o mais especﾃｭfica, abaixo segue alguns outros comandos e as explicaﾃｧﾃｵes dos parﾃ｢metros:

GIF com tamanho final especﾃｭfico e com loop:

Detalhes:

  • -r = especifica o frame rate do gif
  • -loop_output = especifica quantas vezes o gif vai entrar em loop (zero para infinito)
  • -s = especifica o tamanho do gif em pixels

GIF com inicio definido pelo tempo e final limitado por frames:

ffmpeg -i widescreen.avi -pix_fmt rgb24 -r 10.0 -loop_output 0 -vframes 100 -ss 00:00:02 -f gif teste.gif

Detalhes:

  • -ss = especifica a posiﾃｧﾃ｣o inicial do video para gravaﾃｧﾃ｣o (formato: hh:mm:ss)
  • -vframes = limita a quantidade de frames que serﾃ｣o gravados a partir do video original
  • No exemplo, o gif serﾃ｡ formado apenas pelos 100 primeiros frames depois da posiﾃｧﾃ｣o 2 segundos

GIF com inicio definido pelo tempo e final limitado pelo tempo:

ffmpeg -i widescreen.avi -pix_fmt rgb24 -r 10.0 -loop_output 0 -ss 00:00:02 -t 00:00:04 -f gif teste.gif

Detalhes:

  • -ss = especifica a posiﾃｧﾃ｣o inicial do video para gravaﾃｧﾃ｣o (formato: hh:mm:ss)
  • -t = especifica a posiﾃｧﾃ｣o final de gravaﾃｧﾃ｣o do video (formato: hh:mm:ss)
  • No exemplo, o gif serﾃ｡ criado a partir do segundo 2 atﾃｩ o segundo 4 do vﾃｭdeo.
